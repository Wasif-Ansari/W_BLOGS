# W-Blogs
A full-stack MERN application. For frontend I have used REACT .For backend I have used NodeJS & database used is MongoDB.


>Functionalities:
* REGISTER
* LOGIN & LOGOUT
* CRUD operations on BLOG POST
* CRUD operations on User
